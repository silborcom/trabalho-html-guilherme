1. O que é HTML:

HTML é uma linguagem de marcação usada para criar páginas na internet.
Com ela é possível organizar textos, imagens, links e outros elementos que aparecem nos sites.
Ele serve como a base de praticamente todas as páginas web.

2. Como surgiu o HTML:

O HTML foi criado no início dos anos 90 por Tim Berners-Lee.
Ele surgiu com o objetivo de facilitar o compartilhamento de informações entre pesquisadores.
Com o tempo, acabou se tornando o padrão principal para criação de páginas na internet.

3. O que é XHTML:

XHTML é uma versão mais organizada e rigorosa do HTML.
Ele segue as regras do XML, exigindo que o código seja bem estruturado e sem erros.
Isso ajuda a tornar as páginas mais padronizadas.

4. Diferença entre XHTML e HTML:

A principal diferença é que o XHTML exige mais regras na escrita do código.
No XHTML todas as tags devem ser fechadas corretamente e bem organizadas.
Já o HTML é mais flexível e permite alguns erros sem quebrar a página.

5. O que é XML:

XML é uma linguagem usada para armazenar e organizar dados.
Ela não serve para mostrar conteúdo na tela, mas sim para estruturar informações.
É muito utilizada para troca de dados entre sistemas.

6. Diferença entre HTML e XML:

HTML é usado para exibir informações em páginas da internet.
XML é usado para armazenar e transportar dados de forma organizada.
Ou seja, HTML mostra o conteúdo e XML organiza o conteúdo.
